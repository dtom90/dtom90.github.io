### Startracker
[GitHub: Startracker](https://github.com/dtom90/Startracker)

* Interactive 3D satellite tracker with AI insights using LLM and Retrieval Augmented Generation.
* **Tech Stack:** React, Next.js, three.js, IndexedDB, Python, FastAPI, Langchain, Gemini, Docker, GitHub Actions

### HeroChat
